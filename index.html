Portfolio Case Study

Automated KYC Verification System

FinTech Azure Logic Apps Power BI BPMN

The Challenge

The client operated on a legacy banking platform where KYC verification was manual and paper-heavy. This resulted in slow onboarding, high human error risk, and stakeholders spending 40+ hours/month compiling reports.

The Approach (BA Lifecycle)

  • Discovery: Workshops with Compliance/Ops teams to define regulatory needs.
  • Modeling (BPMN): Mapped "AS-IS" bottlenecks vs. "TO-BE" event-driven architecture.
  • Requirements: Authored FRDs for Azure Logic App triggers (auto-approve if confidence > 90%).

The Solution

Implemented a Serverless Cloud Solution using Microsoft Azure. Logic Apps orchestrated the workflow between customer portals and the ERP, while Power BI provided live C-suite dashboards.
